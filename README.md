## 🔬 Research Experience

### 1. Base-Station-Free Cooperative Positioning System Fusing UWB & INS
*May 2025 – Apr 2026* | **Role**: Team Leader

Led full-stack development of a multi-node cooperative positioning system, including hardware schematic & PCB design, embedded firmware development, upper-computer & web interface implementation, algorithm design and system debugging.

**Technical Highlights**:
- Developed real-time **ZUPT** (Zero Velocity Update) algorithm to suppress INS drift and improve long-term stability
- Proposed **MDS**-based anchor-free auto-alignment algorithm with ambiguity elimination
- Implemented **UKF**-based tightly-coupled UWB/INS fusion, greatly reducing positioning error
- Built a complete prototype achieving **centimeter-level multi-node continuous positioning**

**Tech Stack**: ESP32-C6, STM32, BMI088, ADXL355, FreeRTOS, UWB/IMU Fusion, Electron, MATLAB

### 2. Trajectory-Oriented Dynamic Autonomous Wearable Positioning System
*Jun 2025 – Nov 2025* | **Program**: Jilin University Innovation & Entrepreneurship Practice Program | **Role**: Co-leader

Conducted research on wearable positioning technology focusing on trajectory-driven dynamic autonomous localization, contributed to system architecture design and algorithm development.

### 3. Deep Learning Based Multi-scenario PDR
*Jan 2026 – Present* | **Role**: Project Leader

Leading the overall design and development of a deep learning enhanced Pedestrian Dead Reckoning system. Covering hardware adaptation, algorithm research, model deployment and system verification across diverse walking scenarios.
